# Fintekk

Fintekk is a modern fintech landing page built with Vite, React, and Tailwind CSS. It showcases a polished product experience with sections for services, how it works, FAQs, testimonials, and newsletter signup.

## Features

- Responsive landing page design
- Interactive FAQ accordion
- Mobile-friendly navigation
- Login and create account views
- Clean component-based React structure

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the development server
   ```bash
   npm run dev
   ```

3. Build for production
   ```bash
   npm run build
   ```

## Project Structure

- src/components - reusable page sections and UI components
- src/App.jsx - main app layout and state
- src/index.css - Tailwind styles and global styling
